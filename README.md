## Description
lab-bot is a Test Framework that allows you to be able to control instrumentation. It is implemented in C# and can be useful for automating instrument control.

## Build
### Dependencies
- Visual Studio 2010
- Castle.Core v3.3.0
- TestStack.White v0.13.3
- SpecFlow v 1.9.0
